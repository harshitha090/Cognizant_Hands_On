# TDD Calculator Assignment

## Objective

Implement a Calculator using the Test-Driven Development (TDD) approach.

## TDD Workflow

### 🔴 Red
Write a failing unit test before implementing functionality.

### 🟢 Green
Implement the minimum code required to make the test pass.

### 🔵 Refactor
Improve the code while ensuring all tests continue to pass.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Exception handling
- Unit testing using PyTest

## Run

Install dependencies

```bash
pip install -r requirements.txt
```

Run tests

```bash
pytest
```
